La idea

El logo es el propio corral

Sin metáfora. La marca es lo que la asociación custodia: un muro bajo de piedra seca, curvado sobre la franja intermareal, que retiene el agua cuando baja la marea. Piedra, agua, línea de marea. Tres trazos.

El muro (piedra seca) · el agua retenida · la línea de marea
Los "cantos" del arco = piedra seca real, no un trazo continuo
Funciona en positivo y en negativo con el mismo peso
Logo

Bloques de firma

El logo horizontal es el uso por defecto. El vertical, para espacios estrechos o centrados. El isotipo suelto, solo cuando el nombre ya está presente (avatar, favicon, sello).

Implementación

Bloque de tokens

Este es el :root canónico. Pégalo verbatim en cada HTML del proyecto; a partir de ahí solo cambia el copy entre idiomas.

/* Los Corrales de Rota — Monumento */
:root{
  /* paleta */
  --sal:#F4F1E9; --roca:#23201C;
  --pizarra:#35494E; --piedra:#9A8F7C; --oxido:#A85A32;
  /* semánticos */
  --bg:var(--sal); --surface:#FCFAF4;
  --text:var(--roca); --text-soft:#6a6357;
  --line:rgba(35,32,28,.12); --brand:var(--pizarra); --accent:var(--oxido);
  /* radios */
  --r-sm:6px; --r-md:12px; --r-lg:18px; --r-pill:999px;
  /* tipos */
  --f-display:'Cinzel',serif;   /* capitales: marca y titulares fuertes */
  --f-body:'Spectral',serif;   /* secciones y texto */
}

Fuentes: Google Fonts (Cinzel, Spectral). En producción, servir en local con @font-face según el patrón del proyecto para no depender de una petición externa.
